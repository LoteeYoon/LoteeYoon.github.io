## C++ Code

```C++
#include <iostream>
using namespace std;

int main()
{
	char ch;
	int N;
	int sum = 0;
	cin >> N;

	for (int i = 0; i < N; i++)
	{
		cin >> ch;
		sum += ch - '0';
	}

	cout << sum << endl;
}
```
