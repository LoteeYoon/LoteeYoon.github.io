## C++ Code

```C++
#include <iostream>
using namespace std;

int main()
{
    int N;
    int layer = 1;

    cin >> N;
    N--;

    while(N > 0)
    {
        N -= layer*6;
        layer++;
    }

    cout << layer << endl;
}
```
