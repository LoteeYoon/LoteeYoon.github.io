## C++ Code

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
	string s;
	string alpabet = "abcdefghijklmnopqrstuvwxyz";
	cin >> s;

	for (int i = 0; i < alpabet.length(); i++)
	{
		if (s.find(alpabet[i]) == string::npos)
			cout << -1 << " ";
		else
			cout << s.find(alpabet[i]) << " ";

	}
}
```
