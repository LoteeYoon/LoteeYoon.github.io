## C++ Code

```C++
#include <iostream>
using namespace std;

int main()
{
    int A, B, C;
    int cost;
    cin >> A >> B >> C;

    cost = C - B;

    if(cost <= 0)
      cout << -1;
    else
      cout << A / cost + 1;
}
```
